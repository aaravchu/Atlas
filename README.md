# Atlas

Atlas is a portfolio-quality AI planner for ambitious college students. It turns messy inputs like goals, class deadlines, and recruiting notes into a clean daily execution system.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm run test
npm run type-check
```

## Environment

- `OPENAI_API_KEY`: required for live planning
- `OPENAI_MODEL`: optional model override for production, defaults to `gpt-4.1-mini`
